# inDriver Auto-Accept Bot 🚗⚡

## الفكرة
تطبيق Android يراقب طلبات inDriver ويقبلها تلقائياً قبل المنافسين.

## المميزات
- ✅ Auto-accept للطلبات الجديدة
- ⚡ سرعة فورية في القبول
- 🎯 فلترة حسب (مسافة، سعر، منطقة)
- 📊 إحصائيات وأرباح
- 🔔 إشعارات للطلبات المقبولة

## طريقة العمل
1. **Accessibility Service** — يراقب notifications و UI بتاع inDriver
2. **أو VPN/Proxy** — يعترض API calls
3. **أو Root + Xposed** — يعدل على التطبيق مباشرة

## المتطلبات
- Android 7.0+
- تفعيل Accessibility Service
- أو Root (للطريقة المتقدمة)

## Tech Stack
- Kotlin / Java
- Accessibility Service API
- Room Database
- Retrofit (API calls)
